# Urban farming｜IELTS 核心词汇（可直接背诵）

## A. 城市农业与生产方式
- urban farming / urban agriculture：城市农业
- rooftop farm：屋顶农场
- soil-free approach：无土方式/无土方案
- soil-based：土壤种植的
- vertical farming：垂直农业
- aeroponic：气雾培的
- growing towers：种植塔
- inner-city farm：内城/市中心农场
- in production：投产中/进入生产阶段
- harvest：收获（动词/名词）
- yield：产量（同等 yield / the same yield）

## B. 资源效率与可持续
- use fewer resources：消耗更少资源
- practical constraints：现实/操作层面的限制
- unused space：闲置空间
- rely solely on：完全依赖于
- organic nutrients：有机营养物质
- closed circuit：闭环系统（循环回路）
- recycled / recycle：回收再利用/循环使用
- exhaust (soil)：耗竭（土壤被耗尽）
- intensive farming：集约化农业
- greenhouse gases：温室气体
- for the same yield：在相同产量下
- consume a tiny fraction of：只消耗极小一部分（能耗表达）

## C. 食品供应链与经济
- food supplies：食物供应/粮食供给
- face threats to：面对……的威胁
- locally / local：本地地/本地的
- transport and storage chain：运输与仓储链条
- refrigerated kilometres：冷链运输里程
- withstand：承受/经受
- wholesaler：批发商
- transport companies：运输公司
- producers：生产者
- field enquiries：处理/应对咨询请求
- design, build and equip：设计、建造并配套（装备/设施）

## D. 描写类高分词（阅读定位/写作加分）
- striking：引人注目的
- bear fruit：结果/开始有产出
- intensely flavoured：味道浓郁的
- resplendently：艳丽地/灿烂地
- sprout：发芽/长出
- abundantly：大量地/丰富地
- hollow：空心的
- dangling：悬垂的
- aromatic herbs：芳香草本
