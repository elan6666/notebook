# scGPT 论文实验详细总结

## 一、论文核心目标

scGPT 想证明的是：

> 一个经过大规模预训练的 single-cell generative foundation model，  
> 可以通过 fine-tuning 迁移到多个下游任务。

论文覆盖的任务包括：

- cell type annotation
- perturbation prediction
- reverse perturbation prediction
- batch correction / integration
- multi-omic integration
- gene regulatory network inference
- transfer learning analysis

---

## 二、实验总览

scGPT 主要做了以下几类实验：

1. 细胞类型注释实验  
2. 扰动预测实验  
3. 反向扰动预测实验  
4. 多 batch scRNA-seq integration 实验  
5. 多组学整合实验  
6. 基因调控网络 / gene program 实验  
7. 迁移学习 / 预训练收益实验  

---

## 三、细胞类型注释实验

### 1. 实验目的

验证：

> scGPT 预训练得到的表示，能否迁移到 cell type annotation 任务？

### 2. 对比方法

- scBERT
- TOSICA

### 3. 数据集

- Mye.
- M.S.
- hPancreas

### 4. 指标

- Accuracy
- Precision
- Recall
- MacroF1

### 5. 含义

这部分主要在证明：

- scGPT 学到的 cell embedding 具有分类语义
- 预训练知识可以迁移到标准注释任务

---

## 四、扰动预测实验

### 1. 实验目的

验证：

> 已知 seen perturbations 后，scGPT 能否预测 unseen perturbation responses？

### 2. 数据集

- Adamson：87 个单基因 perturbations
- Norman：105 个单基因 + 131 个双基因 perturbations

### 3. 数据划分

- 单基因：测试 perturbations 在训练中不可见
- 双基因：分为 0/2、1/2、2/2 unseen genes 三类难度

### 4. 对比方法

- GEARS
- CPA

### 5. 指标

- corr
- corr(Δ)

其中 corr(Δ) 表示相对 control 的表达变化相关性。  
两个指标都在 top 20 DE genes 上计算。

### 6. 结果

论文报告：

- scGPT 在 Adamson 和 Norman 上都取得最高相关性

### 7. 额外分析

论文还做了两类补充验证：

- 在 Adamson 上展示 top 20 DE genes 的真实/预测表达趋势
- 在 Norman 上把全部可能 perturbation 组合扩展到 5565 个，并做 UMAP 聚类，验证预测空间中的 functional grouping

### 8. 说明了什么

说明 scGPT 不只是逐基因数值拟合，  
还学到了一定程度的 perturbation manifold 结构。

---

## 五、反向扰动预测实验

### 1. 实验目的

验证：

> 给定一个结果 cell state，scGPT 能否反推出最可能的 source perturbation？

论文称之为：

> in-silico reverse perturbation prediction

### 2. 数据与设定

- 从 Norman 数据中选 20 个基因构成子空间
- 总共有 210 种单/双基因 perturbation 组合
- 其中 39 个训练、3 个验证、7 个测试

### 3. 方法

把所有 perturbation condition 的预测响应作为 reference database，  
用真实 query cells 去做 topK retrieval，并采用两轮 ensemble voting。

### 4. 结果

论文报告：

- Top-5 内找回正确 perturbation 的比例为 71.4%（5/7）
- 相比在 210 种组合里随机尝试，作者估计可节省约 95.2% 的试错量

### 5. 说明了什么

说明 scGPT 不仅能做前向预测，还能支持：

- target cell state design
- driver gene discovery
- perturbation experiment planning

## 六、多 batch scRNA-seq integration 实验

### 1. 实验目的

验证：

> scGPT 能否在 batch integration 中同时保留 biological variance 并去除 technical batch effect？

### 2. 对比方法

- scVI
- Seurat
- Harmony

### 3. 数据集

- COVID-19
- PBMC 10K
- Perirhinal Cortex

### 4. 指标

#### 生物保真度
- NMIcell
- ARIcell
- ASWcell
- AvgBIO

#### batch correction
- ASWbatch
- GraphConn
- Overall

### 5. 结果

论文指出：

- 在 PBMC 10K 上，scGPT 是唯一能清晰分开所有 cell types 的方法
- AvgBIO 达到 0.821，比对比方法高 5–10%

### 6. 说明了什么

说明 scGPT 学到的表示不仅适合生成任务，也适合 integration embedding。

---

## 七、多组学整合实验

### 1. 实验目的

验证：

> 统一的 foundation model 是否能跨不同 omics 模态学习可对齐表示？

### 2. 实验类型

包括：

- paired RNA + ATAC：10x Multiome PBMC
- paired RNA + Protein：BMMC
- mosaic RNA + ATAC + Protein：ASAP PBMC

### 3. 对比方法

- scGLUE
- Seurat v4
- scMoMat

### 4. 说明了什么

这是 scGPT 区别于很多专用 perturbation 模型的重要部分：

- 它不只做 perturbation
- 还想成为统一 single-cell / multi-omics backbone

---

## 八、基因调控网络 / gene program 实验

### 1. 实验目的

验证：

> scGPT 的 gene embeddings 和 attention maps 是否隐式编码了 gene-gene interaction 以及 condition-specific activation pattern？

### 2. 展示内容

Figure 5 中包括：

- HLA gene network
- CD gene network
- cell-type specific gene program activation
- Reactome pathway enrichment
- 与 co-expression 方法对比

### 3. 说明了什么

说明 scGPT 的预训练表示具有一定可解释性，  
不只是黑盒分类器或预测器。

---

## 九、迁移学习 / 预训练收益实验

### 1. 实验目的

验证：

> 预训练是否真的有助于多个下游任务？

### 2. 结果

论文总结指出：

- fine-tuned scGPT 在 cell type annotation、multi-batch integration、multi-omic integration 等任务上， consistently 优于 from-scratch 模型

### 3. 说明了什么

这说明：

- 预训练知识确实可以迁移
- foundation model 路线在 single-cell 场景下是有效的

---

## 十、一句话总结

scGPT 的实验逻辑是：

> 先证明它在多个标准下游任务上都能工作，  
> 再证明它在 perturbation prediction 上也有竞争力，  
> 然后进一步展示 reverse perturbation、multi-omics integration、gene network inference 等更高层能力，  
> 最终论证：scGPT 不是单任务模型，而是一个可以“预训练后按需微调”的通用 single-cell foundation model。
