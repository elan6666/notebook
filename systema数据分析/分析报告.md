# 基于 Systema 的系统性变异分析与 scPRAM-VAE 潜空间表征：开源单细胞扰动数据的机器学习分析报告

  
> 数据来源：<Norman2019 / Adamson2016 / 

---

## 摘要
本项目针对开源单细胞遗传扰动数据集，复现并扩展 Systema 框架中“systematic variation（系统性变异）”的分析思想，重点研究不同 reference（control centroid vs perturbed centroid）对扰动几何结构的影响。同时，引入 scPRAM 的 VAE 编码器作为核心机器学习方法，学习细胞的潜在表示（latent embedding），并在潜空间中开展聚类、UMAP 可视化与批次效应分析，比较原表达空间与潜空间下的 systematic variation 差异。此外，本文设计 value binning（表达值离散分箱）作为跨数据集统一表达尺度的特征工程模块，评估其对跨数据集可比性与聚类结构的影响。

**关键词**：single-cell perturbation；Systema；systematic variation；reference choice；VAE；scPRAM；latent space；value binning；UMAP；batch effect

---

## 1. 引言
### 1.1 背景
- 单细胞扰动数据用于研究基因扰动/药物处理对细胞转录组的影响。
- 传统评估或分析常以 control centroid 为 reference，可能混入“跨扰动一致的系统性偏移”，导致对扰动特异信号的高估。

### 1.2 本项目目标
1. 基于 Systema 的实现，计算并可视化数据集的 systematic variation，比较不同 reference 的敏感性。
2. 使用 scPRAM 的 VAE（encoder）学习潜空间表示，在潜空间中进行聚类、UMAP、批次分析，并对比原空间与潜空间的 systematic variation。
3. 引入 value binning 统一表达尺度，作为跨数据集整合/对比的特征工程模块（可选对照实验）。

### 1.3 贡献与创新点（课程角度）
- 不是复现一个预测模型，而是构建“可复用分析管线”：reference-sensitive 的扰动几何分析 + VAE 潜空间机器学习 + 常规单细胞探索分析 +（可选）表达尺度统一。

## 2. 数据集与预处理
### 2.1 数据集
- 数据集名称：Norman2019 / Adamson2016 / ...  
- 数据类型：Perturb-seq（多扰动条件 + control）
- 数据规模：细胞数 N、基因数 G、扰动条件数 P
- 获取方式：
  - 方案 A：调用 systema-main/src/data.py（内部使用 GEARS PertData）自动下载与切分
  - 方案 B：直接使用本地 .h5ad（已含 obs: condition/control）

### 2.2 关键字段标准化（必须写清楚）
保证 AnnData.obs 至少包含：
- `condition`：字符串；control 条件统一记为 `"ctrl"`
- `control`：0/1（1 表示 control）
- （可选）`batch` / `donor` / `replicate`：用于批次分析

### 2.3 预处理流程
- QC：过滤低基因数/低 UMI/高线粒体比例（阈值写清楚）
- 标准化：normalize_total → log1p
- 特征选择：HVG（例如 2k/3k/6k）
- 表达矩阵格式：根据训练需要可转 dense（注意内存）

### 2.4 Value Binning（可选模块：跨数据集统一尺度）
定义：将每个细胞的表达值映射到离散等级（0 表示 0，非零值按分位/区间分箱到 1..B）。
- 目的：减少跨平台/跨数据集表达尺度漂移，使“等级语义”更一致。
- 对照方案：
  - A：continuous log1p 表达
  - B：binned 表达（作为 VAE 输入或作为聚类输入的替代特征）

## 3. 方法
### 3.1 Systema：systematic variation 与 reference 选择
#### 3.1.1 centroid 与 shift
对每个扰动条件 p：
- `c_p`：该条件细胞平均表达（centroid）
- `c_ctrl`：control centroid
- `c_pert_avg`：所有扰动条件 centroid 的平均（perturbed centroid）

定义两种 reference 下的 shift：
- control reference：Δ_p^ctrl = c_p - c_ctrl
- perturbed reference：Δ_p^pert = c_p - c_pert_avg

#### 3.1.2 systematic variation（核心指标）
定义平均扰动效应向量：
- avg_shift = c_pert_avg - c_ctrl

对每个扰动条件 p，计算：
- sim_p = cosine(Δ_p, avg_shift)

systematic variation 取全体扰动的平均：
- SV = mean_p(sim_p)

> 直观：若大多数扰动 shift 都与 avg_shift 同向，则 SV 高，说明系统性偏移强。

#### 3.1.3 在原表达空间 vs 潜空间的 Systema 分析
- 原表达空间：AnnData.X 为基因表达
- 潜空间：AnnData.X 为 VAE encoder 输出 latent embedding（维度 D_latent）

在两种空间中都重复计算：
- SV_ctrlref、SV_pertref
并对比其分布与差异。

---

### 3.2 scPRAM-VAE：机器学习表征学习模块
使用 scPRAM 官方实现的 `SCPRAM`（本质为 VAE）：
- 输入：细胞表达向量（G 维）
- 输出：latent embedding（D 维）

训练目标（简述）：
- 重建损失 + KL 正则（VAE）

训练完成后，通过 `get_latent_adata(adata)` 得到 latent AnnData：
- latent_adata.X：N × D
- latent_adata.obs：拷贝原 obs（保留 condition/control/batch）

---

### 3.3 潜空间下的常规单细胞分析
在 latent_adata 上做：
- PCA / Neighbors / UMAP
- 聚类：KMeans（课程 ML 友好）与 Leiden（单细胞友好）
- 批次分析（若存在 batch）：
  - UMAP 按 batch 上色
  - silhouette score（batch vs biological label）
  - （可选）kBET/LISI 等

## 4. 实验设计
### 4.1 实验 1：reference-sensitive 的 systematic variation
- 空间：原表达空间
- reference：control vs perturbed centroid
- 输出：SV 分布、箱线图/直方图、pairwise cosine 分布

### 4.2 实验 2：VAE 潜空间是否改变 systematic variation
- 空间：latent space（scPRAM-VAE）
- reference：control vs perturbed centroid
- 输出：SV 对比（原空间 vs 潜空间）

### 4.3 实验 3（可选）：value binning 对跨数据集一致性的影响
- 输入：continuous vs binned
- 分析：
  - UMAP/聚类结构变化
  - batch mixing 的变化
  - SV 的变化
- 结论：binning 是否提升跨数据集可比性/鲁棒性
